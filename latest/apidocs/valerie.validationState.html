<static> findIn(model, includeSubModels, recurse, validationStates)

Finds and returns the validation states of:
  • immediate properties of the given model
  • immediate sub-models of the given model, if specified
  • descendant properties of the given model, if specified
  • descendant sub-models of the given model, if specified
Parameters:
Name Type Argument Default Description
model object the model to find validation states in
includeSubModels boolean <optional>
 true whether to return the validation states of child sub-models
recurse boolean <optional>
 false whether to inspect the descendant properties and, if specified, descendant sub-models of child sub-models
validationStates array.<valerie.IValidationState> <optional>
 the already inspected validation states; this parameter is used in recursive invocations
